# MyWidget

MyWidget is a simple Android application that demonstrates how to build and configure a custom **App Widget** using Android Studio.  
It includes a widget provider, manifest configuration, and resource files to display a widget on the home screen.

---

## 📱 Features
- Custom home screen widget (`MyWidgetProvider`)
- Configurable widget layout (`widget_layout.xml`)
- Resource management (colors, themes, strings)
- Gradle build setup with Kotlin support
- Example unit and instrumented tests

---

## 🛠 Project Structure
- `app/src/main/java/` → Kotlin source code (e.g., `MainActivity.kt`, `MyWidgetProvider.kt`)
- `app/src/main/res/` → XML resources (layouts, drawables, mipmaps, values)
- `app/src/main/AndroidManifest.xml` → Widget provider configuration
- `gradle/` → Gradle wrapper files
- `build.gradle.kts` → Project build configuration
- `app/build.gradle.kts` → Module build configuration
